投資結論

SIVE 的投資價值不該被簡化成「CW laser 供應商」。 它更接近 CPO / ELS 架構遷移中的上游設計點供應商。市場若用傳統 pluggable 零組件的 TTM revenue 與 P/S 去看，會低估 qualification cycle 帶來的延遲價值；若直接把它說成全市場 chokepoint，則會高估其控制力。

看多基礎

Sivers 已經被放進 Ayar、POET、O-Net / Enablence、Jabil 這幾條 CPO / ELS 或 1.6T LRO 路線。這些合作不是單純報價單，而是產品驗證、模組整合、ODM 與量產準備。

需折扣

CPO 與 ELS 架構：為什麼外部光源變重要

AI cluster 的資料流量從 GPU 內部、機櫃內、機櫃間，逐步推高互連頻寬與能耗壓力。傳統 copper 與可插拔光模組仍有位置，但在高密度 switch ASIC / GPU package 附近，電路損耗、散熱、面積與 serviceability 會形成硬限制。CPO 的方向是把光學元件靠近 compute / switch package，降低電氣路徑損耗；ELS 的方向則是把最怕熱、最需要波長穩定的雷射移出高熱區。

  A[GPU / Switch ASIC 高熱源] --> B[CPO 光子晶片或 optical engine]

  D[External Light Source 模組] -->|穩定 CW 光源| B
  E[DFB laser arrays] --> D
  F[Star coupler / WDM / PLC] --> D
  G[ODM 與封裝組裝] --> D
  D --> H[可維修、可替換、降低熱耦合]
  B --> I[高頻寬、低能耗、短電氣路徑]

為什麼不用把雷射直接塞進 package

雷射對溫度敏感。高熱會造成波長漂移、效率下降、可靠度壓力與維修困難。CPO 把光引到 package 附近，但 ELS 把光源放在外部，解決的是熱與維修問題。

為什麼不是任何 CW laser 都能用

CPO / ELS 需要的是可被系統架構吸收的光源：功率、波長穩定、陣列一致性、耦合效率、封裝尺寸、散熱路徑、可靠度與供應穩定性同時成立。

雷射技術拆解：CW、DFB、EML 不在同一個比較維度

CW laser：連續光源，不等於低價 commodity

CW laser 的字面意思是 continuous-wave laser。它只描述光源連續輸出，不代表架構、波長數、功率密度、熱設計或封裝相容性。把 CW laser 當成「隨便換」是錯誤簡化。

DFB laser array：Sivers 目前論述的核心

Distributed Feedback（DFB）雷射的價值在於波長控制與可整合陣列。對 CPO / ELS 來說，關鍵不是單顆雷射規格，而是多波長、多通道、低熱負擔、可靠封裝與可被 optical engine 吸收。

EML：另一條光模組主戰場

Electro-Absorption Modulated Laser（EML）把雷射與調變能力放在一起，主戰場常見於高速 pluggable 與資料中心光模組。Lumentum 與 Coherent 在 EML 有深厚能力，但 EML 強不代表 ELS / CPO 架構中所有 DFB array 設計點都可直接取代。

ELS module：從零件進入架構層

ELS 是把高功率雷射陣列、波長分配、耦合、封裝與 ODM 組裝放進一個可部署模組。這也是 Sivers 從單一 laser die 敘事往下游 optical engine / module TAM 擴張的關鍵路徑。

技術本質主要價值誤判點對 SIVE 的意義
CW laser連續波光源提供穩定光功率把 CW 當成全部都可互換只是入口，不是護城河本身
DFB laser array分佈回饋雷射陣列波長穩定、多通道、可整合只拿單顆功率比規格目前最重要的技術卡位
EML電吸收調變雷射高速調變與 pluggable 出貨把 EML 供給等同 ELS 供給競爭背景，非同一設計點
ELS外部光源模組隔離熱源、提升維修性、支援 CPO把模組當成單顆雷射SIVE 往 TAM 擴張的主要載具

供應鏈與瓶頸：SIVE 是哪一種 chokepoint

上游晶圓與磊晶
III-V 材料、InP / GaAs、晶圓製程與良率。這層決定可用 die 數量與長期成本曲線。
雷射設計與陣列
Sivers 目前卡在這層。價值在高精度雷射陣列、DFB、波長穩定與客製化設計。
ELS / optical engine
POET optical interposer、Enablence NxN Star Coupler、O-Net ODM、Ayar optical I/O 等，把光源變成系統可用模組。
模組與系統
Jabil 1.6T LRO、pluggable / LPO / CPO 路線、hyperscaler qualification、量產測試與供應鏈簽約。
最精準的說法：SIVE 是「特定 CPO / ELS 架構的供應鏈配額瓶頸」，不是「整個光通訊雷射市場的永久瓶頸」。當客戶進入 qualification 並預留產能，SIVE 對該設計點有 pricing power；當多供應來源成熟，這個 power 會下降。

競爭格局：Lumentum、Coherent 與日系供應商仍是壓力

Lumentum（LITE）

具備 laser chips、EML、narrow linewidth laser assemblies、pump lasers、transceivers 等完整能力。近期法說內容顯示 EML 與 laser chip 供給緊張，Japan wafer fab fully allocated，並提到 ultra-high-power laser chip ramp for CPO。

強競爭者同時驗證產業瓶頸

Coherent（COHR）

在高速光通訊與 EML 技術長期深耕，展示 400G differential EML。它證明大型供應商有能力推進高速光源，但這不代表 Sivers 已有的客製 DFB array design-in 會被立即取代。

強競爭者替代威脅

Furukawa / Sumitomo 等日系供應商

具備光通訊元件與高可靠度供應能力。它們是多供應來源策略下最自然的替代候選，但替代速度取決於架構相容、qualification、封裝與客戶風險承受度。

多供應來源壓力

Sivers（SIVE）

優勢在特定架構的早期設計點、客製高功率 DFB laser array，以及與 CPO / ELS 新創和 ODM 的結合。劣勢在規模、資本支出承受力、量產良率與供應鏈議價。

卡位稀缺規模折扣

財務模型：用 milestone discounting，不用成熟 P/S

這類公司在 2024 到 2026 的財報通常看不出真正的上行，因為價值在設計導入、產品驗證、預採購、產能分配與量產準備。正確模型是把每個技術與商業節點拆成條件機率，再折現到 2027 之後的量產收入。

風險與驗證節點

技術風險

高功率雷射陣列的熱、波長穩定、耦合效率、封裝可靠度與量產一致性。任何一項失敗，都會讓 design-in 回到 prototype 階段。

商業風險

客戶導入節奏延後、多供應來源成功、hyperscaler 更換架構、ODM 或 optical engine 夥伴改變路線。

資本風險

量產前需要資本支出與營運資金。若客戶不願以長約或預付款支撐擴產，Sivers 的議價能力會被資本能力限制。

最該追的五個訊號
  1. 客戶把 prototype 轉成 production readiness 或正式採購。
  2. 出現 capacity reservation、pre-purchase、long-term agreement。
  3. Sivers 披露 DFB laser arrays 或 ELS 相關出貨量指標。
  4. 毛利率開始反映量產 mix，而不是開發收入。
  5. 主要客戶沒有把第二供應來源推到同等 qualified status。
