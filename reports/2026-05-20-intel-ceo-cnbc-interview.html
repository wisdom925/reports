重點摘要： 本次專訪釋出五大可驗證信號——14A 製程量產時程、EMIB-T 封裝機會、CPU 需求三倍跳升、18A 良率提前達標，以及代工客戶承諾指標（CapEx）。每一項均可作為後續追蹤 Intel 轉型進度的量化錨點。

五大核心信號

信號 1｜代工客戶驗證指標
CapEx = 真客戶

Tan 親自定義：只有當資本支出上升、設備採購增加，才代表有實際簽約客戶。這是過去 CEO 從未給出的可追蹤指標。

信號 2｜18A 良率提前達標
年底前完成 → Panther Lake 量產

原訂年底達標，實際提前完成。外部客戶已主動詢問開放接單時間，良率每月改善 7-8%。

信號 3｜CPU 需求暴增
客戶預測量 ×3

有客戶要求今年訂單增加三倍，Agentic AI 驅動 CPU:GPU 比例從 1:8 翻轉至 1:4，甚至 1:1 或 4:1。

信號 4｜14A 製程對齊 TSMC
2028 風險量產 / 2029 HVM

14A（1.4nm）時程與 TSMC N2 對齊，0.5 PDK 已提供多位客戶，EMIB-T 先進封裝瞄準 CoWoS 缺口。

信號 5｜基板客戶預付款
客戶已預付基板費用（罕見承諾）

基板供應鏈緊俏，部分客戶主動預付以確保物料，這是半導體業中相當強烈的客戶綁定訊號，顯示需求不是口頭承諾。

CPU 需求爆發：比例翻轉

AI 工作負載演進驅動 CPU:GPU 比例持續上升，Tan 引用客戶數據

* 4:1 為部分客戶提及的極端情境

驅動力
  • → Inference 推理：大量平行請求，CPU 是調度核心
  • → Agentic AI：多 Agent 協調管理需 CPU 處理編排邏輯
  • → 強化學習（RLHF）：環境模擬與獎勵計算 CPU 密集
  • → 訓練→推理轉移：推理佔比提升，整體 CPU 需求上升

Tan 認為需求持續性

CPU 短缺不是一次性衝擊，而是結構性轉變。Agentic AI 工作負載本質上是 CPU 密集，這與市場過去重 GPU 的敘事大幅不同。Intel 是全球主要 x86 CPU 廠商，這個缺口直接轉化為 Xeon 訂單。

TSMC CoWoS 現況
  • • 全球 AI 晶片封裝需求爆發
  • • TSMC 產能持續供不應求
  • • 客戶等待期拉長，影響 AI 基礎設施部署
Intel EMIB-T 機會
  • • 下一代先進封裝（次於 CoWoS 競品）
  • • 目標：達到可靠量產良率
  • • Tan 自評：「擁有最好的技術」
  • • 瞄準 CoWoS 溢出需求作為切入點

Tan 的策略是先不正面挑戰 TSMC，而是承接其無法滿足的訂單——這是一個務實的市場進入策略，風險較低但可建立規模與信譽。

客戶預付基板費用——罕見的強烈信號

在半導體供應鏈中，客戶預付材料費用意味著他們相信需求是真實且緊迫的，願意承擔資金風險來確保供應優先權。這通常發生在產能極度緊張的時期（如 2020-2022 晶片荒）。

對 Intel 的意義
  • • 需求可見度提升，降低庫存風險
  • • 客戶沉沒成本增加切換障礙
  • • Tan 可更有信心投資供應鏈
潛在風險
  • • 基板供應鏈擴產週期長（18-24個月）
  • • 若需求反轉，Intel 仍面臨超額庫存
  • • 競爭對手 AMD 同樣受益於需求上升

代工客戶驗證邏輯
「隨著時間推移，我們的 IP 會準備就緒，能夠服務這些客戶。我認為最好的指標是，當你看到我增加資本支出、投入資金購買設備，那就代表我有真正的客戶。這是我堅持的紀律。」
14A 製程時程
「（14A）就是 1.4nm，這是最先進的製程。老實說，2028 年我們將進入風險生產，2029 年進入量產。這與 TSMC 的時程完全一致，這是一項重大的突破，我非常興奮。我們已經有多位客戶與我們接洽，而且我們已提供 0.5 PDK。」
CPU 需求與比例翻轉
「目前如我所說，CPU 需求非常強勁，這對我來說是好事。我甚至無法足量出貨給客戶。過去訓練時 CPU 對 GPU 的比例是 1:8，現在因為 Inference、Agentic AI、更多 Agent 需要管理、協調，以及強化學習，CPU 的重要性大幅提升，比例變成 1:4、甚至 1:1，有人甚至提到 4:1，這對我來說是推動 CPU 業務的巨大機會。」
18A 良率提前達標
「另外就是良率表現與缺陷密度，本來預計年底達到目標，現在甚至在年底前就達標了——這給我很大的鼓舞，也因此 Panther Lake 現在就能量產出貨。現在有些客戶主動來敲門說，Lip Bu，我們聽說你們進展非常好，現在可以開放給外部客戶了嗎？這真的很令人興奮。」
組織改革：打破孤島
「從我第一天擔任 CEO 開始，我就讓所有工程師直接向我報告，這樣我才能真正了解問題、聽到客戶聲音，並知道錯誤出在哪裡。」

「公司過去有太多孤島、太多層級的人彙報……所以我決定，最好的方式就是真正找出問題所在，專注於工程、重新設計、簡化產品，然後推出真正殺手級的產品。」
New Intel 願景
「我招募了一些關鍵人才……到今年 6 月底，我將擁有自己的團隊，我們會一起規劃未來 5 年、10 年的方向，讓 Intel 成為一家不同的公司。我稱之為『New Intel』，以光速前進，團隊合作，一起向前邁進。」
