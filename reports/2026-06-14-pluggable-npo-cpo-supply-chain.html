在 AI 資料中心帶動的 800G / 1.6T 升級周期中，光通訊模組正從可插拔 (Pluggable) 型態演進到近封裝 (NPO) 與共封裝 (CPO) 方案。這篇報告整理技術架構差異、供應鏈分層、生態廠商，以及台灣 PCB / 半導體 / 散熱廠商的間接曝險路徑。

核心觀點：Pluggable 仍是 2025-2027 年主流交付型態；NPO 是過渡路線；CPO 是 2028 年以後長期方向。對台灣供應鏈而言，最先落地的價值增量在有機基板 (Organic Substrate)、高多層 PCB、散熱模組與鑽石散熱材料，而非立即取代既有光模組廠。

名詞定義與技術代際

為避免混淆，本報告採用業界最常用定義：Pluggable 指可插拔光模組 (如 QSFP-DD / OSFP)，NPO 指 Near Package Optics（光引擎靠近封裝基板但仍在封裝體外部），CPO 指 Co-packaged Optics（光引擎直接封裝在 Switch ASIC 同一條封裝線內）。

技術代際定義技術特徵預計量產窗口代表方案
Pluggable可插拔光模組，以電連到主機板標準的外插模組型態，易維護、易升級當前主流QSFP-DD 800G、OSFP 800G
NPO光engine靠近封裝體但仍獨立物理存在縮短 golden bonding length，降低功耗與延遲2025-2027 過渡期Broadcom / NVIDIA 發表中的 NPO 方案
CPO光引擎直接與 Switch ASIC 共封裝最大程度縮短 optical-electric 轉換距離2027+ 小批量，2029+ 規模化Broadcom / Intel / 台積電 CoWoS 衍伸路線

演進驅動力：功耗密度提升、SerDes 損耗、訊號完整性、co-packaging 所需散熱 / 封裝材料技術成熟。

台廠間接曝險路徑

台廠大多不是直接生產 CPO / NPO 光引擎，而是從 PCB、基板、散熱、封裝測試切入。以下按曝險等級排序：

曝險等級標的類型代表標的 / 說明受益邏輯
高載板 / ABF 基板 / 玻璃 core南電、景碩、欣興、臻鼎NPO / CPO 均需更高密度有機基板與玻璃基板驗證
高高多層 / 高頻 PTFE PCB台光、金居、燿華、定穎50G+ PAM4 銅通道對 low-loss 材料需求提升
中高封裝代工 (OSAT)日月光、矽品、力成NPO 階段已有合封封裝需求
中散熱材料 / 熱管 / 風扇雙鴻、奇鋐、泰碩、光鋏CPO 模組功耗密度極高，散熱成瓶頸
間接網通設備代工 / 測試智邦、廣達、緯創、英業達上游 AI server / switch 訂單周期帶動
間接砷化鎵 / 磷化銦磊晶全新、曜鴻、光燿雷射器 chip 台廠代工，受益幅度取決於是否進入 CPO 供應鏈
注意：以上曝險路徑基於公開產業邏輯推斷，不等同公司已確認訂單或技術到位。台廠是否真實受惠，仍需追蹤實際 product qualification 進度。

趨勢、風險與不確定因素

CPO 量產時程可能再次延後

OIF / IEEE 的 connector / PMD 標準仍在制定中，供應商擔心過早押注單一路線，導致實際放量時程可能延後至 2029 年。

Co-packaging 良率挑戰

光引擎與 ASIC 直接封裝對封裝廠的熱處理、hybrid bonding、clean room 等級都要求極高，初始良率若過低會影響客戶導入意願。

標準化競爭

CPO 目前有 OIF、MDF、EDSFF 等多種路線競爭，connector / mechanical standard 若分裂，會導致供應鏈碎片化。

美中貿易與出口管制

高速 AI 晶片與先進封裝技術受到出口管制影響，若限制擴大，可能會影響台廠的設備與材料來源，進而拖慢 CPO 本地化生產節奏。

散熱的滾雪球效應

CPO 的功耗集中度大幅提升，若散熱方案遲遲未能成熟，會直接造成 ASIC 無法 safe operate，進而打亂 adoption 曲線。

結論

短期 (2025-2026)：Pluggable 800G 仍是主力規格，供應鏈仍以傳統光模組廠為主。台灣間接受惠於高速 PCB (50G PAM4)、散熱與高頻材料需求增加。

中期 (2026-2028)：NPO 方案開始小量導入，有機基板與 CoWoS-like 封裝需求增加，日月光、台積電的 2.5D 封裝產能成為 bottleneck。台廠載板 / 基板廠需通過 glass core 與 fine-pitch 的能力驗證。

長期 (2028+)：CPO 成為 AI 交換器主流，台廠受益機會最大的是材料 / 基板 / 散熱 / OSAT 代工，但前提是標準定型、良率達標、供應鏈本地化通過驗證。

核心結論：台廠在 CPO / NPO 世代不是被動旁觀者，而是材料與製造重鎮；但受益順序是「基板 → 散熱 → 封裝 → 器件」，不是反過來。
