# Image Specifications

- **Type**: Infographic
- **Layout**: dashboard
- **Style**: technical-schematic
- **Aspect Ratio**: 16:9 (Landscape)
- **Language**: Traditional Chinese (正體中文)

## Core Principles
- 遵循精密工程圖面與儀表板（Dashboard）的排版架構。
- 採用深藍色（Deep Blue, #1E3A5F）背景與細網格線，搭配發光亮藍、青色（Cyan）以及橘黃色（Amber, #F59E0B）的數據高亮。
- 資訊精準、高對比，突顯 2奈米製程升級帶來的需求變革。

## Layout Guidelines (dashboard)
- 整體呈現多個數據小工具（Widgets）與 KPI 指標卡片。
- 頂部放置大標題：「昇陽半導體 (8028) 2奈米與先進封裝擴產新週期」。
- 畫面分為三大主要區塊：
  1. **營運動態與產能面板 (KPI Cards)**：放置營收數字、單季 EPS 與產能目標。
  2. **製程升級需求對比圖 (Demands Chart)**：以長條圖或刻度線展示 28nm -> 3nm -> 2nm 的再生晶圓需求倍數變化。
  3. **成長引擎分析區 (System Analysis)**：列出自動化規模經濟、晶圓薄化、在地供應。

## Style Guidelines (technical-schematic)
- 科技藍圖（Blueprint）風格：深藍底、白色或亮藍細線線條、幾何精確圖案。
- 文字使用乾淨的無襯線字體，關鍵指標採用大字級高亮。
- 適度加上標記線（Dimension lines）與工程符號（e.g. 坐標、百分比環狀圖、向上箭頭）。
