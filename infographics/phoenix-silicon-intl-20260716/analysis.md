# 昇陽半導體 (8028) 產業分析 Infographic 分析報告

## 1. 內容分析 (Content Analysis)
- **主題**：昇陽半導體 (8028) 受益於台積電 2奈米製程擴產及先進封裝需求，迎來高階再生晶圓與晶圓薄化的成長週期。
- **數據類型**：包含產能預估（85萬片 -> 120萬片）、財務表現（Q1 營收 13.03 億、Q2 營收 14.27 億、EPS 1.61元）、製程技術對再生晶圓需求之倍數比較。
- **語調與讀者**：專業投資人、半導體產業研究者。語調需客觀、結構清晰、數據高對比。

## 2. 設計目標 (Design Goal)
- 將四大成長因子與財務/產能數據進行高密度的視覺化排版，使讀者能在 30 秒內掌握昇陽半導體的核心長線投資價值。
- 特別強調 2奈米 GAA 技術帶來 3 倍再生晶圓需求變化的數據對比。

## 3. 語言設定
- **繁體中文** (Traditional Chinese)
- **台灣用語**優先 (e.g., 再生晶圓、晶圓薄化、先進封裝、台積電、製程、產能、稼動率)。
